Data Science

Duration: 8 weeks · Level: Beginner → Intermediate

Course Overview

This course provides a comprehensive introduction to the field of Data Science, combining theory, tools, and hands-on projects. Students will learn how to collect, analyze, and interpret data using Python, SQL, and modern machine learning techniques. By the end, learners will be able to turn raw data into actionable insights and build simple predictive models.

Curriculum

  1. Introduction to Data Science
  2. Programming for Data Science (Python)

UI/UX Design

Duration: 8 weeks · Level: Beginner → Intermediate

Course Overview

This course teaches the full process of designing user-centered digital products. Learners will explore user experience (UX) principles, user interface (UI) design techniques, and prototyping tools to build intuitive, visually appealing, and accessible applications. By the end of the course, students will be able to design complete app or website interfaces — from user research to high-fidelity prototypes.

Curriculum

  1. Introduction to UI/UX Design
  2. Visual Design (UI)
  3. User Research & Empathy
  4. UX Writing & Microcopy
